# MMDT Data Engineering Program

Welcome to the **MMDT Data Engineering Program** repository!  

This repository serves as a comprehensive resource hub for participants, including:

- **Projects**: Hands-on assignments to apply your data engineering skills to real-world scenarios.  
- **Tutorials**: Step-by-step guides to build your knowledge and confidence in data engineering tools and techniques.  
- **Course Materials**: Lecture notes, exercises, and examples to support your learning journey.

### Purpose
This is a public repository designed for **participants to share, collaborate, and submit homework** while fostering a community of learning and growth.
